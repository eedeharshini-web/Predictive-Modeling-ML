# Predictive Modeling Using Machine Learning

## Project Overview

This project demonstrates predictive modeling using Machine Learning techniques. The Iris flower dataset was used to train and test a Random Forest Classifier model for predicting flower species based on input features.

## Tools and Libraries Used

* Python
* Scikit-learn
* Matplotlib
* Seaborn

## Tasks Performed

* Loaded and explored the dataset
* Split the dataset into training and testing sets
* Trained the Random Forest Classifier model
* Predicted outcomes using test data
* Evaluated model accuracy
* Visualized results using a confusion matrix and heatmap

## Machine Learning Algorithm Used

* Random Forest Classifier

## Visualization

A confusion matrix heatmap was generated to analyze prediction performance and classification accuracy.

## Outcome

This project helped in understanding:

* Supervised learning concepts
* Predictive modeling techniques
* Model training and testing
* Accuracy evaluation
* Confusion matrix visualization
